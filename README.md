## 四、智能预算调控模型
### 4.1 动态计算公式

基准预算 = 历史均值×(1+0.3×CTR增长率)
周调整量 = 前3天均值×(1+0.1×转化趋势)


### 4.2 学习期保护机制
- 前72小时锁定预算
- 固定5组核心受众
- 采用3版标准素材
- 固定广告版位
